# Robin Singh Rajawat — Portfolio

Personal portfolio and resume site, built as a single static HTML file.

**Live site:** [robinrajawat.github.io/portfolio](https://robinrajawat.github.io/portfolio/)

## About

SAP EWM Solution Architect — experience, skills, and certifications, plus
[Sakura](https://robinrajawat.github.io/sakura), a personal productivity project.

## Structure

- `index.html` — the entire site (markup, styles, and content in one file)
- `robin-photo.jpg` — portrait used in the hero section
- `resume/` — resume (`.docx` and `.pdf`) and LinkedIn cover image
- `favicon.ico`, `favicon-16x16.png`, `favicon-32x32.png`, `apple-touch-icon.png` — site icons
